# 剪贴板助手 - 官方网站

这是剪贴板助手（Clipboard Assistant）Mac应用的官方支持网站，包含营销页面、技术支持和隐私政策。

## 网站结构

本网站包含以下页面：

### 1. 首页 (index.html)
营销页面，展示应用的所有核心功能：
- 自动剪贴板管理
- 快速搜索
- 多种复制触发方式
- 历史记录上限可调
- 注重隐私保护
- 安全可靠

### 2. 技术支持 (support.html)
提供全面的技术支持信息：
- 常见问题解答 (FAQ)
- 系统要求
- 联系方式
- 故障排除指南

### 3. 隐私政策 (privacy.html)
符合 Apple App Store 上架要求的完整隐私政策：
- 详细说明不收集任何用户数据
- 辅助功能权限使用说明
- 符合 GDPR、CCPA 和 Apple 审核指南
- 本地存储和加密说明

## GitHub Pages 部署

### 启用 GitHub Pages

1. 进入仓库的 Settings
2. 找到 "Pages" 设置
3. 在 "Source" 部分选择要部署的分支（通常是 `main` 或 `master`）
4. 保存设置

几分钟后，你的网站将在以下地址可用：
```
https://[你的用户名].github.io/clipboard_support/
```

### 自定义域名（可选）

如果你有自己的域名，可以在 Pages 设置中配置自定义域名。

## 自定义说明

### 更新联系邮箱

在部署前，请更新以下文件中的联系邮箱：

1. **support.html** (第 101-102 行)
   - 将 `support@example.com` 替换为你的实际支持邮箱

2. **privacy.html** (第 178 行)
   - 将 `privacy@example.com` 替换为你的实际隐私政策联系邮箱

### 自定义内容

你可以根据实际需求修改：
- 应用名称和描述
- 功能列表
- 系统要求
- 联系方式
- 隐私政策内容

### 样式自定义

所有样式都在 `styles.css` 文件中，你可以修改：
- 颜色方案（在 `:root` 变量中定义）
- 字体
- 布局
- 响应式断点

## 技术栈
