# Clipboard Safe - 官方网站

这是Clipboard Safe（Clipboard Assistant）Mac应用的官方支持网站，包含营销页面、技术支持和隐私政策。

## 网站结构

本网站包含以下页面：

### 1. 首页 (index.html)
营销页面，展示应用的所有核心功能：
- 自动剪贴板管理
- 快速搜索
- 多种复制触发方式
- 历史记录上限可调
- 注重隐私保护
- 安全可靠

### 2. 技术支持 (support.html)
提供全面的技术支持信息：
- 常见问题解答 (FAQ)
- 系统要求
- 联系方式
- 故障排除指南

### 3. 隐私政策 (privacy.html)
符合 Apple App Store 上架要求的完整隐私政策：
- 详细说明不收集任何用户数据
- 辅助功能权限使用说明
- 符合 GDPR、CCPA 和 Apple 审核指南
- 本地存储和加密说明

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- 移动浏览器

## 许可证

本项目供Clipboard Safe应用使用。
